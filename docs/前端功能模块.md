# LQRAgent 前端功能模块说明（学生工作台）

> **给谁看**：负责 **学生端** 前端的同学。  
> **写什么**：学生工作台要做什么页面与组件（接口字段和测试请访问 Swagger：http://localhost:8080/swagger-ui.html）。  
> **不含**：管理后台 UI —— 由后端同学自行开发维护，能力清单见 [后端开发进度 §三](./后端开发进度.md)。  
> **最后更新**：2026-05-22

## 一、产品边界

| 本文档覆盖（学生端） | 不在本文档（另见后端进度 / 自研 Admin） |
|--------------------|----------------------------------------|
| 登录后进入 `/workspace` 的学习体验 | `/admin` 管理面板布局与 Tab |
| 聊天、路径、资源、答题、画像、上传 | 模型配置、队列运维、用户管理等后台界面 |
| 多模态资源展示（图 / 视频） | 用户自助注册 |

| 明确不做（相对 DeepTutor 开源 UI） |
|-----------------------------------|
| 不嵌入 `ai-server/web` |
| 不做协作写作、图书编辑器、演练场 |
| 浏览器不直连 `localhost:8001` |

## 二、页面与路由（仅学生端）

```
/login                          登录
/workspace                      工作台 · 聊天（默认）
/workspace/upload               工作台 · 上传资料
/workspace/profile              工作台 · 学习画像
```

管理员登录后跳转 `/admin` 的界面 **不在本文档范围**；现有 `frontend/src/pages/AdminPage.tsx` 仅为占位/参考，可按你方后台方案替换或移除。

| 路由 | 模块编号 |
|------|----------|
| `/login` | M1 |
| `/workspace/*` | M2～M8 |

## 三、工作台布局（左右两栏，对齐 DeepTutor）

**壳子**：`flex` 左窄侧栏 + 右 `flex-1` 主区（`features/workspace/shell/WorkspaceShell`）。

- **左侧**：品牌、用户卡片、路由导航（聊天 / 上传 / 画像）。
- **右侧主区**：按子路由切换整页；**不使用第三栏**。

**聊天页主区内 Tab**（`context/MainTabBar`）：对话 | 学习路径 | 资源 | 练习。

```
┌──────────┬────────────────────────────────────┐
│ 侧栏      │ 主区                                │
│ · 用户    │  [对话][路径][资源][练习]  ← 仅聊天页 │
│ · 导航    │  当前 Tab 或子路由页面               │
└──────────┴────────────────────────────────────┘
```

| 子模块 | 功能 | 状态 |
|--------|------|------|
| 工作台壳子 | 路由出口 + 可调整布局 | ✅ `WorkspacePage.tsx` |
| 功能导航 | 聊天、上传、画像入口 | ✅ |
| 子路由 | `index` 聊天；`upload`；`profile` | 🟡 profile 占位 |

## 七、按阶段交付（仅学生端）

| 阶段 | 交付 | 验收 |
|------|------|------|
| **P0** | M3 聊天 + `AgentTimeline` | 能流式回复、有智能体步骤 |
| **P1** | M4 真路径 + M5（含示意图） | 点节点能看内容与配图 |
| **P2** | M6 答题 + M8 画像 + 聊天多模态卡片 | 答题后画像有变化 |
| **P3** | M7 上传分析结果展示 | 上传后能看到映射知识点 |
| **P4** | M5 视频、多会话等 | 短视频与体验加分 |

管理端能力（配置、队列、Agent 监控）按 [后端开发进度](./后端开发进度.md) 与你方 Admin 面板自行对齐。
