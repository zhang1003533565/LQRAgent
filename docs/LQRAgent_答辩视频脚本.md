# LQRAgent（Edu.AI）答辩视频脚本

> **用途**：软件杯 / 课程答辩演示视频  
> **时长**：**≤ 7 分钟**（建议成片 6:30～6:50，留剪辑余量）  
> **配套 PPT 脚本**：见 [LQRAgent_答辩PPT脚本.md](./LQRAgent_答辩PPT脚本.md)

---

## 整体时间轴

| 片段 | 时间 | 模块 | 对应 PPT | 形式 |
|------|------|------|----------|------|
| 1 | 0:00～0:40 | 开场 | 1～2 页 | 片头 + PPT |
| 2 | 0:40～1:35 | 架构总览 | 3～4 页 | PPT 录屏 |
| 3 | 1:35～2:50 | 聊天 · 多 Agent | 5～6 页 | **浏览器录屏** |
| 4 | 2:50～3:50 | 路径 · 资源 | 7～8 页 | **浏览器录屏** |
| 5 | 3:50～4:50 | 图谱 · 答题 | 9～10 页 | **浏览器录屏** |
| 6 | 4:50～5:40 | 上传 · 画像 | 11～12 页 | **浏览器录屏** |
| 7 | 5:40～6:30 | 创新总结 | 13 页 | PPT + 片尾 |

**合计：约 6:20～6:30**

## 录制准备清单

- 浏览器打开 `http://localhost:5173`，账号 `student1` / `123456`
- 后端、前端已启动；大模型 Key 已配置
- 浏览器缩放 100%，隐藏书签栏，关闭无关扩展
- 录屏：OBS / Win+G，1920×1080，仅录浏览器窗口
- 提前准备：一条已生成的学习路径、1～2 份已上传课件、1 次答题记录
- 旁白可后期配音；现场旁白语速约 **180～200 字/分钟**

### 分镜操作

| 时间 | 操作 | 旁白稿 |
|------|------|--------|
| +0:00 | 展示侧栏「聊天学习」，指出「已连接」 | 「这是学生工作台，所有 AI 能力从聊天入口触发。」 |
| +0:10 | 输入「你可以干什么」，发送 | 「系统识别为 HELP 意图，直接返回 Agent 能力清单。」 |
| +0:25 | 新建对话，输入「什么是 Python 列表推导式」，发送 | 「答疑走 QA Pipeline，先检索知识库…」 |
| +0:45 | 等待回答完成，**滚动展示引用来源卡片**（若有） | 「…回答附带 RAG 引用，用户可核对依据。」 |
| +0:55 | 输入「我想学 Python，零基础，1 个月」，发送 | 「学习路径请求进入协商流程。」 |
| +1:05 | 展开 Agent 调用链（点击折叠条），快速展示后等折叠 | 「多 Agent 步骤透明可见，完成后自动折叠，只保留结果。」 |
| +1:15 | 展示路径卡片 / 学习路径 Artifact | 「最终输出结构化学习路径，而非纯文本。」 |

**注意**
- 协商较慢时可 **剪入预录成功片段**
- 避免展示失败重试、重复回复等画面

## 剪辑与答辩建议

1. **先录屏、后配音**：演示段静音录操作，旁白单独录，剪辑对齐。
2. **Plan B**：路径/出题生成超过 30 秒，片段 3、4 插入预录素材。
3. **时长控制**：粗剪目标 6:30；超时优先缩短片段 2、7，**不要砍演示**。
4. **Admin 后台**：评委追问配置时可口头说明或展示 PPT 附录第 14 页，主片可不录。
