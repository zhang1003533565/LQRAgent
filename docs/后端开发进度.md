> ⚠️ 以下方法定义在 `AiServerClient.java`，但**真实运行时只有 `ping()` 被调用**。
> 其余方法（createKnowledgeBase / listKnowledgeBases / uploadDocument / getMemory / updateMemory）均为定义态，业务层无调用方，属预留壳子。

| # | 方法 | 实际调 ai-server？ | 调用方 | 备注 |

| 5 | `uploadDocument()` → `POST /api/v1/knowledge/{kb_name}/upload` | ❌ 无人调用 | — | 壳子，等待 P3 |
| 6 | `getMemory()` / `updateMemory()` → `/api/v1/memory` | ❌ 无人调用 | — | 定义态，等待画像同步 |
| 7 | AiServerEnvSyncService 写 .env | ✅ | ModelConfigService | 本地文件写入，非 API 调用 |
| — | ~~`generateQuestion()` / `generateLesson()`~~ | 已删除 | — | DeepTutor 对应路径不存在，改模板兜底 |
