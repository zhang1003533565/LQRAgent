### P4 — 多智能体框架重构（核心架构升级）

目标：从 JVM 直调 → 星型拓扑多智能体系统（Agent 统一契约 + 事件总线 + 硬质量闸门）。

```
[ ] P4-1  Agent 基础契约                  新建 agent/ 包：Agent<Task,Result> 接口 + AgentTask + AgentResult
[ ] P4-2  AgentBus 事件总线               基于 Spring EventBus，Orchestrator 发布→agent 订阅→CompletableFuture 回值
[ ] P4-3  QualityGate 硬质量闸门           所有资源/内容类 AgentResult 强制过 QA，fail 触发 Orchestrator 重试
[ ] P4-4  Orchestrator 改造为派发中心      不再直调 service，改 publish→wait→gate→return
[ ] P4-5  全部 8 智能体迁移至 Agent 接口     每个 agent implement Agent，删除交叉 import
[ ] P4-6  AgentBus 自动可观测性            自动记录每次 process 耗时/成功/失败 → agent-stats 实时更新
```
